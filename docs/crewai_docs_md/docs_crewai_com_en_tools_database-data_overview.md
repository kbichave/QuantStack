[**MySQL Tool** \\
\\
Connect to and query MySQL databases with SQL operations.](https://docs.crewai.com/en/tools/database-data/mysqltool) [**PostgreSQL Search** \\
\\
Search and query PostgreSQL databases efficiently.](https://docs.crewai.com/en/tools/database-data/pgsearchtool) [**Snowflake Search** \\
\\
Access Snowflake data warehouse for analytics and reporting.](https://docs.crewai.com/en/tools/database-data/snowflakesearchtool) [**NL2SQL Tool** \\
\\
Convert natural language queries to SQL statements automatically.](https://docs.crewai.com/en/tools/database-data/nl2sqltool) [**Qdrant Vector Search** \\
\\
Search vector embeddings using Qdrant vector database.](https://docs.crewai.com/en/tools/database-data/qdrantvectorsearchtool) [**Weaviate Vector Search** \\
\\

\\
Vector similarity search on MongoDB Atlas with indexing helpers.](https://docs.crewai.com/en/tools/database-data/mongodbvectorsearchtool) [**SingleStore Search** \\
\\
Safe SELECT/SHOW queries on SingleStore with pooling and validation.](https://docs.crewai.com/en/tools/database-data/singlestoresearchtool)

## [​](https://docs.crewai.com/en/tools/database-data/overview\#common-use-cases)  **Common Use Cases**

- **Data Analysis**: Query databases for business intelligence and reporting
- **Vector Search**: Find similar content using semantic embeddings
- **ETL Operations**: Extract, transform, and load data between systems
- **Real-time Analytics**: Access live data for decision making

Copy

Ask AI

```
from crewai_tools import MySQLTool, QdrantVectorSearchTool, NL2SQLTool

# Create database tools
mysql_db = MySQLTool()
vector_search = QdrantVectorSearchTool()
nl_to_sql = NL2SQLTool()

# Add to your agent
agent = Agent(
    role="Data Analyst",
    tools=[mysql_db, vector_search, nl_to_sql],
    goal="Extract insights from various data sources"
)
```
