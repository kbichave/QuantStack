# [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#bright-data-tools)  Bright Data Tools

This set of tools integrates Bright Data services for web extraction.

## [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#installation)  Installation

Copy

Ask AI

```
uv add crewai-tools requests aiohttp
```

## [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#environment-variables)  Environment Variables

- `BRIGHT_DATA_API_KEY` (required)
- `BRIGHT_DATA_ZONE` (for SERP/Web Unlocker)

Create credentials at [https://brightdata.com/](https://brightdata.com/) (sign up, then create an API token and zone).
See their docs: [https://developers.brightdata.com/](https://developers.brightdata.com/)

## [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#included-tools)  Included Tools

- `BrightDataSearchTool`: SERP search (Google/Bing/Yandex) with geo/language/device options.
- `BrightDataWebUnlockerTool`: Scrape pages with anti-bot bypass and rendering.
- `BrightDataDatasetTool`: Run Dataset API jobs and fetch results.

## [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#examples)  Examples

### [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#serp-search)  SERP Search

Code

Copy

Ask AI

```
from crewai_tools import BrightDataSearchTool

tool = BrightDataSearchTool(
    query="CrewAI",
    country="us",
)

print(tool.run())
```

### [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#web-unlocker)  Web Unlocker

Code

Copy

Ask AI

```
from crewai_tools import BrightDataWebUnlockerTool

tool = BrightDataWebUnlockerTool(

print(tool.run(url="https://example.com"))
```

### [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#dataset-api)  Dataset API

Code

Copy

Ask AI

```
from crewai_tools import BrightDataDatasetTool

```

## [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#troubleshooting)  Troubleshooting

- 401/403: verify `BRIGHT_DATA_API_KEY` and `BRIGHT_DATA_ZONE`.
- Empty/blocked content: enable rendering or try a different zone.

## [​](https://docs.crewai.com/en/tools/web-scraping/brightdata-tools\#example)  Example

Code

Copy

Ask AI

```
from crewai import Agent, Task, Crew
from crewai_tools import BrightDataSearchTool

tool = BrightDataSearchTool(
    query="CrewAI",
    country="us",
)

agent = Agent(
    role="Web Researcher",
    goal="Search with Bright Data",
    backstory="Finds reliable results",
    tools=[tool],
    verbose=True,
)

task = Task(
    description="Search for CrewAI and summarize top results",
    expected_output="Short summary with links",
    agent=agent,
)

crew = Crew(
    agents=[agent],
    tasks=[task],
    verbose=True,
)

result = crew.kickoff()
```
