# [​](https://docs.crewai.com/en/tools/search-research/serpapi-googleshoppingtool\#serpapigoogleshoppingtool)  `SerpApiGoogleShoppingTool`

## [​](https://docs.crewai.com/en/tools/search-research/serpapi-googleshoppingtool\#description)  Description

Leverage `SerpApiGoogleShoppingTool` to query Google Shopping via SerpApi and retrieve product-oriented results.

## [​](https://docs.crewai.com/en/tools/search-research/serpapi-googleshoppingtool\#installation)  Installation

Copy

Ask AI

```
uv add crewai-tools[serpapi]
```

## [​](https://docs.crewai.com/en/tools/search-research/serpapi-googleshoppingtool\#environment-variables)  Environment Variables

- `SERPAPI_API_KEY` (required): API key for SerpApi. Create one at [https://serpapi.com/](https://serpapi.com/) (free tier available).

## [​](https://docs.crewai.com/en/tools/search-research/serpapi-googleshoppingtool\#example)  Example

Code

Copy

Ask AI

```
from crewai import Agent, Task, Crew
from crewai_tools import SerpApiGoogleShoppingTool

tool = SerpApiGoogleShoppingTool()

agent = Agent(
    role="Shopping Researcher",
    goal="Find relevant products",
    backstory="Expert in product search",
    tools=[tool],
    verbose=True,
)

task = Task(
    description="Search Google Shopping for 'wireless noise-canceling headphones'",
    expected_output="Top relevant products with titles and links",
    agent=agent,
)

crew = Crew(agents=[agent], tasks=[task])
result = crew.kickoff()
```

## [​](https://docs.crewai.com/en/tools/search-research/serpapi-googleshoppingtool\#notes)  Notes

- Set `SERPAPI_API_KEY` in the environment. Create a key at [https://serpapi.com/](https://serpapi.com/)
- See also Google Web Search via SerpApi: `/en/tools/search-research/serpapi-googlesearchtool`

## [​](https://docs.crewai.com/en/tools/search-research/serpapi-googleshoppingtool\#parameters)  Parameters

### [​](https://docs.crewai.com/en/tools/search-research/serpapi-googleshoppingtool\#run-parameters)  Run Parameters

- `search_query` (str, required): Product search query.
- `location` (str, optional): Geographic location parameter.
